Washing Machines: From Dumb to Smart

This project was made during the Human and Computer Interaction course that focused a lot on User Experience and smart interfaces. Our project was about making interfaces for washing machines smarter by designing a smart washing machine that can auto decide what program you should use without having to manually click through every option which can be confusing, especially since washing machiens can have very different icons and interfaces from brand to brand.
